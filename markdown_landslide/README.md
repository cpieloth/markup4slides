# README

Generate HTML from Markdown with `landslide`.

- Setup virtualenv
```
$ virtualenv --no-site-packages -p /usr/bin/python3 venv
$ . venv/bin/activate
```
- Install requirements:
```
$ pip3 install -r requirements.txt
```
- Generate HTML:
```
make html
```
- see: https://github.com/adamzap/landslide
